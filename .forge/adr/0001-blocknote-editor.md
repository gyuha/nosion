# 블록 에디터로 BlockNote 채택

노션급 편집 UX(슬래시 메뉴, 드래그 핸들, 블록 변환, 서식 툴바)를 v1에서 확보해야 한다. TipTap 직접 조립은 그 UI 전부를 자체 구현해야 해 공수가 2~3배이고, contenteditable 위 직접 구현은 수개월짜리 별도 프로젝트라, 노션 스타일 UX를 기본 탑재한 BlockNote(ProseMirror/TipTap 기반, React)를 채택했다.

## 결과

- 문서 포맷이 BlockNote의 블록 JSON 스키마에 종속된다. 에디터를 교체하려면 저장된 전체 문서의 마이그레이션이 필요하다.
- XL 패키지(멀티컬럼, PDF/DOCX 내보내기 등)는 AGPL/상용 이중 라이선스라 사용하지 않고 코어(MPL-2.0)만 쓴다. 노션의 다단 레이아웃은 이 제약으로 비목표가 된다.
