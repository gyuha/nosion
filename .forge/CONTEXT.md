# Nosion

노션(Notion)을 로컬 환경에서 클론한 문서 도구. 계정별 독립 워크스페이스에서 블록 기반 문서 편집과 테이블 데이터베이스를 제공한다.

## 용어

**워크스페이스**:
한 사용자의 모든 페이지가 담기는 최상위 컨테이너. 가입 시 자동으로 하나 생성되며 사용자 간 공유되지 않는다.
_피할 표현_: 스페이스, 계정

**페이지**:
사이드바 트리의 노드이자 생성·이동·삭제의 단위. 문서 페이지와 데이터베이스 페이지 두 종류가 있으며 무한히 중첩된다.
_피할 표현_: 노트, 문서(단독 사용)

**문서 페이지**:
블록의 나열을 본문으로 갖는 페이지.

**데이터베이스 페이지**:
속성 스키마와 행 목록을 본문으로 갖는 페이지.
_피할 표현_: 테이블 페이지, DB

**블록**:
문서 본문을 구성하는 최소 편집 단위(문단, 헤딩, 리스트, 코드 등).
_피할 표현_: 노드, 엘리먼트

**속성**:
데이터베이스의 열 정의. 타입은 텍스트·숫자·셀렉트·다중 셀렉트·날짜·체크박스.
_피할 표현_: 필드, 컬럼

**행**:
데이터베이스의 항목 하나. 열어 보면 그 자체가 문서 페이지다.
_피할 표현_: 레코드, 아이템, 로우

**휴지통**:
소프트 삭제된 페이지의 보관소. 복원하거나 영구 삭제할 수 있다.
_피할 표현_: 아카이브
